Opie Taylor : Minecraft Datapack

This fantastic and somewhat overpowered datapack will give you the ability to craft some things you wish you could.

Download opie.zip

Installing

To install this datapack download the datapack file opie.zip and then copy it to the datapacks directory in one of your worlds (at ~/.minecraft/saves/{worldname}/datapacks. See the datapack installation tutorial for more details.

What it does

Opie crafting table updates

The following are some Opie crafting recipes to enjoy.

Craft you some leather!

Use that annoying leftover rotten flesh to make some leather.

Craft you a saddle!

Craft a saddle from two leather and an iron ingot.

Craft you some blaze rods!

Craft a blaze rod by using a torch, a gold ingot and a stick.

Craft some netherite scrap!

Netherite scrap is actually made of three nether brick blocks, two flint, one basalt, one calcite, one soul sand and some raw copper in this shapeless recipe.

Craft you some elytra!

You can now craft an elytra from white wool, two sticks, four feathers and a clay ball.

Craft you a shulker shell!

Craft a shulker shell from two eggs, a purple carpet, three quartz and two amethyst shards.

Opie stonecutter updates
